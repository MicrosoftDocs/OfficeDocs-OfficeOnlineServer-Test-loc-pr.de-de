# Technische Referenz für Office Web Apps Server

 

_**Gilt für:**Office Web Apps Server_

_**Letztes Änderungsdatum des Themas:**2016-12-16_

**Zusammenfassung:** Enthält Links zu technischen Referenzartikeln für Office Web Apps Server, z. B. Hilfe zu den Windows PowerShell-Cmdlets vom Typ "OfficeWebApps".

**Zielgruppe:** IT-Spezialisten

![Office 2013-Logo](images/Ee890081.a106e261-2cd0-43b7-af77-92de7e4b6fb9(Office.15).png "Office 2013-Logo") Office Web Apps Server ist ein neues Office-Serverprodukt, das browserbasierte Versionen von Word, PowerPoint, Excel und OneNote bereitstellt. Eine einzelne Office Web Apps Server-Farm kann Benutzer unterstützen, die über SharePoint 2013, Lync Server 2013, Exchange Server 2013, freigegebene Ordner und Websites auf Office-Dateien zugreifen. In der folgenden Tabelle werden technische Referenzartikel für IT-Spezialisten aufgelistet und beschrieben, die Office Web Apps Server lokal in Ihren Organisationen bereitstellen.
